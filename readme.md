# Jupyter Reader for Acode

A lightweight extension that allows you to preview and read Jupyter notebooks (.ipynb files) directly in Acode mobile editor. This is a read-only viewer - you can view the notebook but cannot edit it.

## Features

- **Read-only Preview**: View .ipynb files without the ability to edit
- **Cell Types Support**: Displays code cells, Markdown cells, and raw cells
- **Markdown Rendering**: Markdown cells are rendered with basic formatting
- **Code Cell Display**: Shows code with input prompts and execution counts
- **Output Display**: View cell outputs including text, errors, HTML, and images
- **Lightweight**: Minimal dependencies, fast loading

## What You CAN Do

- Open and preview any .ipynb file
- View code cells with syntax and execution counts
- Read rendered Markdown cells
- View cell outputs (text, images, errors)
- See notebook metadata (kernel info, cell count)

## What You CANNOT Do (Read-Only)

- Edit cell content
- Add or delete cells
- Run code cells
- Save changes to the file
- Modify the notebook in any way

This is intentional - the plugin is designed for viewing notebooks without risk of accidental modifications.

## Usage

1. Open any `.ipynb` file in Acode
2. The notebook will automatically render in the viewer
3. A "Read-only mode" notice will be displayed at the top
4. Scroll through cells to view content and outputs

## Requirements

- Acode Editor v290+

## Installation

1. Download the plugin
2. In Acode, go to Settings > Plugins
3. Click the `+` button
4. Select "Local" and choose the downloaded zip file
